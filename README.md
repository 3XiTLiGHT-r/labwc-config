# labwc-config

## Installation

Copy configs:

```sh
mkdir -p ~/config/labwc
cp autostart  ~/config/labwc/
cp environment  ~/config/labwc/
cp menu.xml  ~/config/labwc/
cp rc.xml  ~/config/labwc/
```
